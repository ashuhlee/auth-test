# AuthKit Login Test
AuthKit, Vite + Express.js demo for local login testing.

### Build from Source
1. Clone this repository:
```zsh
git clone https://github.com/ashuhlee/auth-test.git
cd auth-test
```
2. Install dependencies:
```zsh
npm install
```
3. Create a `.env` file in the project root using `.env.example`:
```zsh
cp .env.example .env
```

4. Run the app in development mode:
```zsh
npm run dev
```

App runs at `http://localhost:3000`<br>
[AuthKit documentation](https://workos.com/docs/authkit)
